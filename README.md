# Rock-paper-scissor-game
A simple Python console game where you play Rock, Paper, Scissors against the computer.
This project uses if-else statements, ASCII art, and Python’s random module.

How to Play

Run the Python script.

Enter a number to choose your move:

0 → Rock

1 → Paper

2 → Scissors

The computer will randomly choose its move.

The winner is announced: win, lose, or draw!
